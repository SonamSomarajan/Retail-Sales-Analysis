# Retail-Sales-Analysis

Objective: 
Analyze sales data to identify revenue trends, customer behavior, and category opportunities.

Tools Used:
1. Excel
2. PowerPoint
3. Word

Files Included:
1. Cleaned Excel file with pivot tables and dashboards
2. Presentation
3. Screenshots of dashboards for sales, customer and product category analysis
4. Project documentation PDF

Key Insights:
1. Electronics drove highest revenue
2. Clothing led sales volume
3. 50+ customers were highest-value customers
4. Seasonal variations impacted performance

Recommendations:
1. Focus electronics growth
2. Improve clothing monetization
3. Target high-value customer segments
